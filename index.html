요구사항 작성

2

Codex 초안 생성

3

사람 검토

개발자가 생성된 코드를 검토하고 문제점을 찾아냅니다.

4

Claude Code 분석

Claude Code가 코드를 분석하여 개선사항을 제안합니다.

5

리팩토링

피드백을 반영하여 코드를 더 깔끔하고 효율적으로 개선합니다.

6

테스트 및 검증

작성된 코드가 제대로 작동하는지 꼼꼼하게 테스트합니다.

7

Git 저장

완성된 코드를 Git 저장소에 저장하여 버전 관리를 합니다.

📖 실행 방법

1단계: 파일 준비

다음 3개의 파일이 같은 폴더에 있는지 확인하세요:

  • index.html - 웹페이지 구조
  • style.css - 디자인 스타일
  • script.js - 기능 구현

2단계: 웹브라우저에서 열기

index.html 파일을 웹브라우저(Chrome, Edge, Safari 등)에서 열어주세요.
방법:

  • 파일 탐색기에서 index.html을 더블클릭
  • 또는 브라우저 주소창에 파일 경로를 붙여넣기

3단계: 앱 사용

  • 이름을 입력창에 입력하세요
  • 「실습 시작」 버튼을 클릭하세요
  • 환영 메시지가 나타나면 성공입니다!
